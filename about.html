02 : Projects

Fleet Management System

My partner, Teg, and I, built the basic functionality of a Fleet Management System for cargo vessels using MERN FullStack. Our software is currently being used by LFonds Pvt Ltd, onboard their 2 cargo vessels, IDM doodle and Danship Bunker. We are currently building extra modules to commercialise the project by adding predictive engine wear systems and predictive fuel bunkering and maintenance systems.

Warehouse Management System

On top of the Fleet Management System (FMS), we built the frontend dashboard of a Warehouse Management System using React JS and will be building the full WMS once we complete our FMS project. We want to provide an integrated supply chain software for companies in the supply chain.

Admin Dashboard

A MERN FullStack Admin Dashboard for an Enterprise Resource Planning System. This will be integrated with the WMS and FMS to create a full Supply Chain Software Ecosystem.

Computer Vision System for Hewlett Packard (HP)

Did this internship during Poly days where we built a OpenCV Python program to quality check printer nozzles in the HP printer production line. This saved the employees time from having to manually check the quality of each printer leaving the HP factory daily.
